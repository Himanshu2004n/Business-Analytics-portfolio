# Product Request Automation (Power Automate)

This project automates product request handling using Microsoft Power Automate, Microsoft Forms, and Excel.

The workflow automatically processes product requests submitted through a form and sends email notifications.

## Workflow Overview

1. A user submits a product request through Microsoft Forms.
2. Power Automate triggers when a new response is submitted.
3. Response details are retrieved automatically.
4. Excel table is accessed to process request information.
5. The flow loops through the relevant records.
6. Email notifications are sent automatically.

## Tools Used

Microsoft Power Automate  
Microsoft Forms  
Excel Online  
Email Automation

## Use Case

Organizations often manage product requests manually.  
This automation improves efficiency by capturing requests and automatically notifying the relevant team.

## Output

Automatic email notification when a product request is submitted.
