# Nykaa Digital Marketing Campaign Performance Dashboard

## Project Overview
This project analyzes Nykaa digital marketing campaigns using Power BI.
The dashboard provides insights into revenue, conversions, marketing funnel performance, and customer segments.

## Tools Used
Power BI
Excel Dataset
Data Visualization

## Key Metrics
Total Revenue: 28.66 Billion
Total Conversions: 57.4 Million
Total Clicks: 260 Million
Conversion Rate: 22.03%

## Dashboard Features
Marketing Funnel Analysis
Campaign Revenue Analysis
Marketing Channel Performance
Customer Segment Insights
