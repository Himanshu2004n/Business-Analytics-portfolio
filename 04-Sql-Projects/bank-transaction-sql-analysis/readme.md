# Bank Transaction Behavior Analysis using SQL

This project analyzes banking transaction data using SQL to identify transaction patterns, customer behavior, and potential risk indicators.

## Dataset
The dataset contains 44,000+ banking transactions including:

- TransactionID
- AccountID
- TransactionAmount
- TransactionDate
- TransactionType
- Channel
- Location
- CustomerAge
- CustomerOccupation
- LoginAttempts
- TransactionDuration

## Key Analysis

The project performs several SQL analyses including:

- Total transaction volume and value
- Transaction type distribution
- Channel-based transaction analysis
- Location-based transaction insights
- Top accounts by transaction value
- Customer spending behavior by age group
- Occupation-based spending patterns
- Detection of suspicious login attempts

## Tools Used

- SQL
- MySQL Workbench
